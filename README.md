# Sales Performance Dashboard - Superstore Analysis

This repository contains a complete analytical study of sales performance between 2014 and 2017, based on Excel pivot tables, dashboards, and detailed insights.

The objective of this project is to identify high-performing areas, detect risks, evaluate annual trends, and support data-driven decision-making.
